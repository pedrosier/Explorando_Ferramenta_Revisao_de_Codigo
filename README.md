# Explorando_Ferramenta_Revisao_de_Codigo
Repositório dedicado à ativida prática 3 da disciplina de Gestão da Qualidade de Software, na qual iremos explorar as funcionalidades e importância de uma ferramenta de revisão de código.
Contém o código-fonte de um sistema de lava-carros desenvolvido em C, utilizado para demonstrar o fluxo de análise e boas práticas via Reviewable.
